Module 01 HTML Basics

***** HTML ***** <-- H1 define the main heading or title of a webpage's content

HTML stands for HyperText Markup Language and is the standard markup language used to create and structure web pages and their content for display in a web browser. It is considered the fundamental building block of the web, working alongside Cascading Style Sheets (CSS) for styling and JavaScript for interactivity.

Key Concepts -->

* Hypertext: Refers to the system of links that connect web pages to one another, allowing users to navigate between related documents and websites by clicking.

* Markup: Involves using a series of elements, known as tags, to annotate or "mark up" content (text, images, etc.) to define its structure and meaning (semantics). The browser interprets these tags as instructions on how to display the content, rather than displaying the tags themselves.

* Semantic HTML: Semantic elements use tags that define the role of the content within them. This provides structural clarity, which is crucial for accessibility (screen readers), search engine optimization (SEO) , and code maintainability. Tag are "header", "nav","article", "section","footer","strong" and "em"

* Non Semantic HTML: Non-semantic elements are generic tags used for structure and presentation, but they do not add any additional meaning or context to the content they wrap. They are primarily used as styling hooks for CSS or for layout purposes when no suitable semantic element is available. Tag are "div","span","i(Italic)","bold"

H2- tag defines secondary headings or subheadings within a webpage's content.

H3- tag defines level-3 type headings within a webpage's content.

H4- tag defines level-4 type headings within a webpage's content.

H5- tag defines level-5 type headings within a webpage's content.
H6- tag defines level-6 type headings within a webpage's content.

* In line elements: An inline element in HTML is an element that does not start on a new line and only takes up as much width as is necessary for its content. The tags "header", "nav","article", "section","footer","strong" and "em" in-line elements.

* Block elements: A block-level element in HTML always starts on a new line and takes up the full width available within its parent container "h1,h2,h3,h4,and so on ", "p" tags are Block level elements

    • *** Division Tag ***

    The HTML "div" tag, short for "division," is a generic container used to group other HTML elements together for styling and layout purposes. It is a block-level element, meaning it typically starts on a new line and takes up the full available width of its parent container by default.

    *** Break tag ***

    The HTML "br" tag (break tag) is used to insert a single line break within a block of text. It is an empty, or void, element, meaning it does not require a separate closing tag.

    New "input" types introduced in HTML5

    1. Color: This input type allows the user to select a color from a color picker.
    2. Date: This input type allows the user to select a date from a drop-down calendar.
    3. Time: This input type allows the user to enter a time.
    4. Datetime: This input type allows the user to select date and time along with timezone.
    5. Week: This input type allows the user to select week and year from the drop-down calendar.
    6. Email: This input type allows the user to enter an e-mail address.
    7. Month: This input type allows the user to select a month and year from a drop-down calendar
    8. Number: This input type allows the user to enter a numerical value.
    9. Range: This input type allows the user to enter a numerical value within a specified range.
    10. Search: This input type allows the user to enter a search string within the input field.
    11. Tel: This input type allows the user to enter a telephone number.
    12. Url: This input type allows the user to enter the URL.

    HTML Semantic Tags for Structure

    1. header: The header tag defines content that should be considered the introductory information of a page.
    2. nav: The navigation tag is used for navigation links
    3. main: This tag contains the main content( also called the body) of a page.
    4. article: The article tag defines content that could stand independently of the page or site it's on.
    5. section: Using "< section >" is way of grouping nearby content of a similar theme.
    6. aside: An aside element defines content that's less important.
    7. footer: You see"< footer >" at the bottom of a page. It usually includes contact information,copyright information and some site navigation.
